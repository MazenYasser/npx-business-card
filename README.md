# npx-business-card

A Claude Code skill that builds and publishes your own interactive terminal business card to npm.

Anyone can run yours with a single command:

```bash
npx mazenx47
```

---

## What it builds

- Animated chalk intro
- Boxen double-border card with your name, title, skills, and links
- Interactive inquirer menu to open GitHub, LinkedIn, or email
- Published to npm so anyone can run it instantly

---

## Install the skill

1. Download [`npx-card.skill`](./npx-card.skill)
2. Open it — Claude Code shows an "Add to library" dialog
3. Click **Add to library**
4. Open Claude Code and say: `Build me an npx card`

---

## Or use the raw skill file

Copy the contents of [`npx-card-skill.md`](./npx-card-skill.md) into a file at:

```
~/.claude/skills/npx-card/SKILL.md
```

Then restart Claude Code.

---

## Live example

